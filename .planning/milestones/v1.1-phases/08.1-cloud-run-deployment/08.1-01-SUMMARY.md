## Decisions Made

1. **No env-gating on PRAGMAs**: Applied unconditionally per D-10 — DELETE journal mode is safe on all filesystems; env-gating would add complexity with no benefit.
2. **Ordering constraint enforced**: DELETE/FULL PRAGMAs placed between `foreign_keys = ON` and `_apply_migrations()`. This prevents WAL materializing during the first DDL `with conn:` transaction on FUSE.
3. **Minimal stub in tests/deploy**: No fixture structure added so Plan 04 can overwrite the body cleanly.
